If any git commands don't work, or if you have any git conflict you must fix that with the console that you can run from the Console menu.

Initialisation

To initialise git, you must choose this during the creation of your project, or add a git repo in your project.

Git add

The classic Git add command that will add on your git the current open file in your IDE.

Git commit

The classic Git commit command that will commit your changes on your git.

Git push

The classic Git push command that will push all your modified files on your git. But be carefull : you will need to enter your credentials to be able to push otherwise it wont work !

Git pull

The classic Git push command that will fetch all the modified files on your repo. But be carefull : you will need to enter your credentials to be able to push otherwise it wont work !

Other usefull command

You can of course use all the other git commands that you want in the console if the button is not implemented yet,feel free to reach us with our contact if you have suggestion ! a list of all usefull git commande can be find here.
