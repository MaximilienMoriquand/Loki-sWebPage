What is Loki

Loki is a simple and easy to use IDE to code in Java. When we created Loki, we wanted to make it as easy as possible to use so that anyone that wanted to use it could without searching hours in the documentation. As we just said, this IDE is made to code and run Java code, but you can of course edit all the documents of your projects.

Who are we ?

We are a group of students that created this IDE as a group project. We based our IDE design to some of our experiences with some tools like Vim or IntelliJ. Thats why we decided to create an very simple IDE that has some IntelliJ features but is as simple as Vim to Use.

Open a project

You can of course open any projects that you want with the IDE and edit any file of your choice. When opening a project, the IDE will automatically detect if this is a Maven project or if your project has some git files. Thats why you won't have any manipulations to do to make the IDE detect that.
