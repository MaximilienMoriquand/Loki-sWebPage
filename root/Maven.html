What is Maven

Maven is a build automation tool used primarily for Java projects.

Maven can also be used to build and manage projects written in C#, Ruby, Scala, and other languages.

The Maven project is hosted by the Apache Software Foundation, where it was formerly part of the Jakarta Project.

feel free to visit this site if you want to know more about maven !

Maven compile

Command used to compile your project to run it or deploy an executable version of it.

Maven run

It is the command that we use in this IDE to run Java code. Be carrful : you won't be able to run your project without Maven.

Maven test

Command used to test your project with Maven.

Maven Package

Command used to make a target package of your project to use it in other projects or deploy it.

Other usefull command
